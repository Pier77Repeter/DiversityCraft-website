What is DiversityCraft?

DiversityCraft is a community that was created back on the 4th of February 2014 as an unique, friendly and chill survival server, with a few close people playing on it. Since it's beginning the server never had any type of moderation or rules to follow, giving the players full freedom (since Minecraft is a sandox game) to play on the server however they like to, without any restrictions. The only limit is your immagination, you are completly free to do whatever you want on the server with no consequences.

Join this awesome server today!
