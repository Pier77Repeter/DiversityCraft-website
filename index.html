What is DiversityCraft?

DiversityCraft is a community that was created back on the 4th February 2014 as a friendly and chill survival server with a few close people playing on it, since it's beginning the server never had any type of moderation or rules to follow giving the players the full freedom (since Minecraft is a sandox game) to play on the server however they like to, without any restrictions the only limit is your immagination, you are completly free to do whatever you want on the server with no consequences.

Join the server today!
