Commands

  • - /help See all the aviable commands
  • - /stats Get informations about the server's world
  • - /msg [player name] Send a private message to the player
  • - /r Reply to the latest private messages that you've received
  • - /ignore [player name] Ignore a player from chat temporarily
  • - /ignorehard [player name] Ignore a player from chat permantly
  • - /ignorelist Shows the list of all players you've blocked
  • - /togglechat Hide/Show the chat
  • - /toggleprivatemessages Hide/Show private messages
  • - /sit You just sit (press shift to stop sitting)

Anonymous chat system

    The server has a system to anonymize your messages in chat:
  • - /anon Anonymizes your messages in chat (ONLY IN PUBLIC CHAT)
  • - /unanon Stop anonymizing your messages in chat
  • - /toggleanons Hide/Show anonymized players in chat
  • - /infoanon Get more informations on how to use this system
