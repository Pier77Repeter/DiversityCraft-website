Gameplay/Gamerules modifications

    Some Minecraft base features have been turned off or modified for a better experience on the server:
  • - Chat reporting is completly disabled
  • - Phantoms spawning has been disabled
  • - The gamerule "enderPearlsVanishOnDeath" has been disabled
  • - Sign editing has been turned off, to prevent "history editing/griefing"
  • - IP logging has been disabled in the server config to provide privacy by default"
  • - You can sit, crawl and spin with a few commands, see here

Voicechat

    The server has voicechat plugin enabled, you need to install the mod for it to work, download here
  • 1 - Before downloading Voicechat make sure you download Fabric loader: here and Fabric API: here
  • 2 - Boot the game with Fabric installed, then close once it's up
  • 3 - Drag your Voicechat mod into the ".minecraft/mods" folder with the Fabric API
  • 4 - If you did everything correctly you are good to go!

Custom tablist

    The server has a custom player list/tablist that displays the following informations:
  • - TPS (Tick-Per-Second) check the server peformance (green = no lag, yellow = kinda laggy, red = lag)
  • - Online players and the max player slots
  • - Your ping to the server (lower is better)
  • - Server's uptime
  • The tablist messages will update for any server news or with a random sentences suggested by the players

Server Anti-Cheat

    While cheating on the server is allowed, some modules have been restricted or disabled to prevent big exploits or hurt the server performances, since we try our best to make the gameplay experience smooth for everyone:
  • - Movement exploit have been disabled to prevent going 1000KM/h, stressing the server's performances
  • - Duplications glitches are patched
  • - Game-breaking exploits that are in the vanilla game have been patched
  • - Non-survival friendly modules have been disabled
